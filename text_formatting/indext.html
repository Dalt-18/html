Web Development Terminologies

Web Development Terminologies

HTML
HyperText Markup Language, used to structure content on the web.
CSS
Cascading Style Sheets, used to style and layout web pages.
JavaScript
A programming language used to create interactive effects in web browsers.
Frontend
The part of a website users interact with directly (HTML, CSS, JavaScript).
Backend
The server-side of a website, handling databases, logic, and authentication.
API
Application Programming Interface; allows communication between software systems.
DOM
Document Object Model; represents HTML as a tree structure for JavaScript manipulation.
Responsive Design
Designing web pages that adapt to different screen sizes and devices.
Framework
A collection of pre-written code used to help develop web applications faster (e.g., React, Angular).
Library
A set of reusable code designed to perform specific tasks (e.g., jQuery).
HTTP
Hypertext Transfer Protocol; the foundation of data communication on the web.
HTTPS
Secure version of HTTP using SSL/TLS encryption for secure data transfer.
SEO
Search Engine Optimization; techniques used to improve website visibility on search engines.
URL
Uniform Resource Locator; the web address used to access resources on the internet.
Cookie
Small pieces of data stored by the browser to remember information about the user.
Session
Data stored on the server to persist user interactions across multiple requests.
Web Hosting
A service that allows websites to be accessible on the internet.
Git
A version control system used to track changes in code.
REST
Representational State Transfer; an architecture style for designing networked applications.
JSON
JavaScript Object Notation; a lightweight format for storing and transporting data.
